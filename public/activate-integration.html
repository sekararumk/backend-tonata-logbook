🎯 Cara 1: Inject Manual (Recommended)

1. Buka halaman Table.js di tab lain: http://localhost:3000/Homepage

2. Buka Developer Tools (tekan F12)

3. Pilih tab "Console"

4. Copy dan paste script berikut:

const script = document.createElement('script'); script.src = 'http://localhost:5001/integration/table-integration.js'; script.onload = () => console.log('✅ Integration script loaded!'); script.onerror = () => console.log('❌ Failed to load integration script'); document.head.appendChild(script);
